# PBR Material Viewer
## NodeJS WebGL2

Physically Based Rendering (PBR) materials are 3D computer graphics shaders and textures designed to accurately simulate how light interacts with surfaces in the real world. They ensure consistent, photorealistic appearances under varying lighting conditions by utilizing maps like Albedo, Roughness, and Metallic.

This material viewer supports parallax heightmap and soft shadows, along with albedo, roughness, metallic, and ambient occlusion maps.

![Demo GIF](https://github.com/arjunsinghgill/pbr-material-viewer/blob/main/pbr_material_viewer.gif)

To run the demo, clone the repository and use NodeJS command:
> node src.pbr_material_viewer.https.js
